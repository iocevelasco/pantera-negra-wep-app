# Assets para Hero

Coloca las siguientes imágenes en esta carpeta:

- `assets-1.jpg` - Imagen 1 para el hero
- `assets-2.jpg` - Imagen 2 para el hero
- `assets-3.jpg` - Imagen 3 para el hero
- `assets-4.jpg` - Imagen 4 para el hero

Las imágenes se mostrarán en las tarjetas del hero de la página de inicio.

## Formato recomendado

- Formato: JPG o PNG
- Resolución: Mínimo 1080x1920px (formato vertical/portrait)
- Tamaño: Optimizado para web (máximo 500KB por imagen)
